KRAKE Quick Start

Basic usage and where to find key diagnostics.

  • Home: serial, MAC, IP, firmware, MQTT status, and live LCD output.
  • Settings: broker/topic settings, mute control, and WiFi reset.
  • Broker Console: publish test alarms and monitor ACK topics.
  • Factory Test / Developer Monitor: live UART monitor output.
  • PMD Web UI: select saved target topics from Settings and send PMD alarm messages.
  • MQTT Device Monitor: track online/offline devices by MAC-based MQTT topics.

Tip: The serial number shown on these pages is the MAC-based device identifier used in topics.
